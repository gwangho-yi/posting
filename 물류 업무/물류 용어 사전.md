|    환적    |             Transshipment             |        T/S        | 운송 중인 화물을 직접 목적지로 보내지 않고 중간 기항지에서 다른 선박(또는 운송 수단)으로 옮겨 싣는 행위. 예: 부산→싱가포르(환적)→LA  |
|    집하    |                Pick up                |        P/U        |       포워더나 운송사가 수출자의 창고나 공장에서 화물을 인수하여 내륙 운송을 시작하는 단계. Door to CY/CFS의 시작점       |
|    혼재    |             Consolidation             |      CONSOL       |              여러 화주의 소량 화물을 모아 하나의 컨테이너에 가득 채우는 작업. LCL 서비스의 핵심 프로세스              |
|   포워더    |           Freight Forwarder           |        FF         |             화물 운송 업무를 대행하는 회사 또는 전문가. 선사, 항공사 등과 계약하여 화주에게 운송 서비스 제공             |
|   운송사    |                Carrier                |         -         | 실제 운송 수단(선박, 항공기, 트럭 등)을 보유하고 화물을 운송하는 회사. 선사(Ocean Carrier), 항공사(Air Carrier) 등 |
|   선적항    |            Port of Loading            |        POL        |           선적항 혹은 적출항이라고 함. 해상 운송 구간에서 화물이 선박에 최초로 실리는 항구. 예: 부산항, 인천항            |
|   도착항    |           Port of Discharge           |        POD        |        양하항, 목적항 혹은 양륙항이라 함. 해상 운송 구간에서 화물이 선박에서 내려지는 최종 항구. 예: LA항, 롱비치항         |
|    화물    |                 Cargo                 |         -         |                  운송되는 상품이나 물품. Goods와 유사하지만 운송 맥락에서 주로 Cargo 사용                  |
|    선적    |          On Board / Shipment          |         -         |                   화물을 선박이나 항공기에 적재하는 행위 또는 그 상태. B/L 발행의 기준 시점                   |
|    양하    |               Discharge               |         -         |                       선박이나 항공기에서 화물을 내리는 행위. Unloading과 동일                       |
|   상업송장   |          Commercial Invoice           |        C/I        |           판매자(수출자)가 구매자(수입자)에게 발행하는 화물의 대금 청구서. 통관 시 필수 서류이며 관세 산정 기준            |
|  구매발주서   |            Purchase Order             |        P/O        |             구매자가 판매자에게 발행하는 공식 구매 요청 문서. 계약서 성격을 가지며 생산 및 운송 계획의 기준              |
|  풀 컨테이너  |          Full Container Load          |        FCL        |                 한 화주의 화물이 컨테이너 한 개를 단독으로 사용하는 방식. CY/CY 조건으로 운송                  |
|    콘솔    |       Less than Container Load        |        LCL        | 여러 화주의 소량 화물을 한 컨테이너에 혼재(Consolidation)하여 운송하는 방식. CFS/CFS 조건. 한국에서는 "콘솔"이라고 통칭  |
|   코로드    |                Coload                 |         -         |          한 포워더가 다른 포워더의 혼재(Consolidation) 서비스를 이용하는 것. Coload B/L이 발행됨           |
|  최초 출발지  |           Place of Receipt            |        POR        |           화물이 포워더에게 인계되는 최초 지점. 내륙 운송 출발지. Door to Door 운송 시 화주의 공장/창고           |
|  최종 목적지  | Final Destination / Place of Delivery | FND / POD(문맥에 따라) |                 화물이 최종적으로 인도되는 지점. Door to Door 운송 시 수입자의 창고/공장                  |
|   발행지    |               Issued At               |         -         |                 B/L, Invoice 등의 서류가 발행된 장소. 예: "Issued at Busan"                 |
|   발행일    |              Issued Date              |         -         |                            B/L, Invoice 등의 서류가 발행된 날짜                            |
| 실제 출발 시간 |       Actual Time of Departure        |        ATD        |                                선박/항공기가 실제로 출발한 시간                                |
| 실제 도착 시간 |        Actual Time of Arrival         |        ATA        |                                    실제로 도착한 시간                                    |
| 예정 출발 시간 |      Estimated Time of Departure      |        ETD        |                                 선박/항공기의 예정 출발 시간                                 |
| 예정 도착 시간 |       Estimated Time of Arrival       |        ETA        |                                예정 도착 시간. 스케줄 계획용                                 |
| 수입 측 비용  |      Revenue / Billing / Selling      |         -         |                         포워더/운송사가 화주로부터 받는 운임 및 비용. 매출 항목                         |
| 지출 측 비용  |        Expense / Buying / Cost        |         -         |                       포워더가 선사, CFS, 트럭 등에게 지급하는 비용. 매입 항목                        |
|   담당자    |           Person In Charge            |       P.I.C       |             B/L, Invoice, 부킹 서류 등에 표기<br>예 : Shipper P.I.C = 화주 담당자,             |
|  수출 화주   |                Shipper                |                   |                                    화물을 보내는 자                                     |
|  수입 화주   |               Consignee               |                   |                                     화물을 받는 자                                     |
